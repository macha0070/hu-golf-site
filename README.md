# ⛳ HU Golf Site
広島大学ゴルフ部公式サイトを作ってみました。
新規メンバーの募集や活動内容の発信を目的に、親しみやすくモダンなデザインを目指して制作しました。

## Demo
hu-golf-site.vercel.app

- **Hero Section**: インパクトのあるメインビジュアルとキャッチコピー
- **About Us**: サークルの活動理念や雰囲気の紹介
- **Gallery**: 活動中の写真や合宿の様子をスライダー形式で表示
- **Contact**: 入部希望者向けのコンタクト情報（またはフォーム）
- **Responsive Design**: スマホ・タブレット・PCすべての画面サイズに対応

## Technologies Used
- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexbox / Grid Layout を使用したレスポンシブデザイン
- **JavaScript**: スライダーやハンバーガーメニューの動作制御

## Concept
「初心者でも入りやすい雰囲気」をコンセプトに、緑（芝生）を基調とした明るい配色にしました。
特にスマホでの閲覧を意識し、読みやすい文字サイズとレイアウトを心がけています。
